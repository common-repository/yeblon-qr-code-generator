== Description ==

Use QR Code in your blog, as a shortcode, a widget or with a PHP function. With this plugin you are able to create QR Codes for your blog posts or other URLs.
<a href="http://yeblon.com/qr-code-generator-wordpress-plugin">Help and more information about QR Code Generator</a>

== Installation ==

1. Upload the full directory into your /wp-content/plugins/ directory, or install through the admin interface
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use as a shortcode (TinyMCE plugin included), as a widget or as a PHP function. <a href="http://yeblon.com/qr-code-generator-wordpress-plugin">Help and more information about QR Code Generator</a>

`<?php yeblonqrcode('100'); // parameter is image size in pixels ?>`
